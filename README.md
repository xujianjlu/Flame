Flame
=====

Construct tools based on Scons

####Example:

  build -- debug model:
 
    flame/make.sh t=test/main [c=dbg]
    
  build -- opt model:
 
    flame/make.sh t=test/main c=opt
    
  clean model:
 
    flame/make.sh t=test/main -c
